## Quick Reference

**Product type:** IoT Hardware + Software (wearable, home sensors, voice assistant + cloud platform)

**Primary users:** Elderly individuals, family caregivers, professional caregivers, healthcare providers

**Deployment:** Elder's private home (aging-in-place)

**Compliance:** HIPAA + general security best practices

**Phase 1 launch target:** Q3 2026
